Truckin' Good

Truckin' good is an application that allows a user to find the location of a food truck and their menu,
read reviews, and rate a truck! The application also allows a food truck to be added to a database.
Technical Toolbelt: HTML/CSS/Javascript/jQuery - Bootstrap - MySql - NPM Sequelize - Twitter API - Heroku- AWS (file uploading)
